## About The Project

Project ini adalah sebuah API sistem managemen hotel sederhana. sistem managemen dapat mengelola Inventory untuk setiap room serta menyimpan setiap riwayat reservasi.

### Built With

* Spring IoC
* Java Stream
* JPQL & HQL
